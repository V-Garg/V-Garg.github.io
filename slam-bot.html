SLAM LiDAR Robot

Project Overview

Role: Lead Engineer

Timeline: June 2024 - August 2024

GitHub:https://github.com/V-Garg/slam_bot

Overview

This project focused on building a SLAM (Simultaneous Localization and Mapping) robot that uses LiDAR to navigate and map its surroundings. As the lead engineer, I was responsible for designing the robot’s structure and implementing a SLAM system using ROS on a Raspberry Pi. The robot utilized custom motor controls with an Arduino, and I integrated a LiDAR sensor for precise environmental mapping and obstacle detection.

Responsibilities

  • Designed and developed the robot’s structure and motor control systems.
  • Implemented SLAM algorithms using ROS on a Raspberry Pi for real-time mapping.
  • Configured the LiDAR sensor for environmental mapping and obstacle detection.
  • Developed a communication interface between the Raspberry Pi and Arduino for seamless control.
